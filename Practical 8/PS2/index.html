CODING CLUB INDIA



Problem Statement: Write a program in any language to read 10 numbers from keyboard and find their sum and average.



Pictorial Presentation:




CODE:

                #include 
                    void main()
                    {       
                        int i,n,sum=0;
                        float avg;
                        printf("Input the %d numbers : \n",n);
                        for (i=1;i<=10;i++)
                        {
                            printf("Number-%d :",i);
                    
                            scanf("%d",&n);
                            sum +=n;
                        }
                        avg=sum/10.0;
                        printf("The sum of 10 no is : %d\nThe Average is : %f\n",sum,avg);
                     
                    }


Variables:

            i - Iterator
            n - Number
            sum - Store sum or numbers
            avg - Store average of numbers


Input:

            
                Input the 10 numbers :                                                                                        
                Number-1 :1                                                                                                   
                Number-2 :2                                                                                                   
                Number-3 :3                                                                                                   
                Number-4 :4                                                                                                   
                Number-5 :5                                                                                                   
                Number-6 :6                                                                                                   
                Number-7 :7                                                                                                   
                Number-8 :8                                                                                                   
                Number-9 :9                                                                                                   
                Number-10 :10

Output:

            
                The sum of 10 no is : 55                                                                                      
                The Average is : 5.500000

Complexity:

            O(n1)
